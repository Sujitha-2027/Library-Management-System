# Library Management System

## Project Description

A Library Management System developed using Python and MySQL. This project allows users to view available books, borrow books, and return books through a simple menu-driven interface.

## Features

* Display available books
* Borrow a book
* Return a book
* Database connectivity using MySQL
* Voice message on exit using pyttsx3

## Technologies Used

* Python
* MySQL
* mysql-connector-python
* pyttsx3

## Database

Database Name: library

Table Name: books
